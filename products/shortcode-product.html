Adding Short Description (Description excerpts)

Use Custom Field and add metafield to metafield sizechart.

You may following instruction about Shopify Metafield name key description_excerpt to add your short product description. You may add html format to Product metafield description excerpt.

<ul><li><span class="a-list-item">Wireless</span></li><li><span class="a-list-item">Motion Sensing</span></li><li><span class="a-list-item">Compatible with Several Consoles/Games</span></li><li><span class="a-list-item">Enough Action Buttons</span></li><li><span class="a-list-item">Ergonomic</span></li></ul>

  • Click Save Custom Fields

Mass edit to add short description c_f.description_excerpt.

  1. Go to: https://shopify.com/admin/bulk?resource_name=Product&edit=metafields.c_f.c_f.description_excerpt:string
  2. Add product short description for each product. . You can add html code to the desription to show list as sample.
